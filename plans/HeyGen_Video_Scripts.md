Implementation Steps

Step 1: Sign up for HeyGen Creator ($24/mo)
- Go to heygen.com
- Select Creator plan
- Access the web studio

Step 2: Create Videos in HeyGen Studio
For each video:
1. Select avatar (same one for all 8 videos for brand consistency)
2. Choose voice per language:
   - EN: Neutral American English
   - ES: Latin American Spanish (es-419)
   - PT: Brazilian Portuguese (pt-BR)
   - DE: German (de-DE)
3. Paste script
4. Generate video
5. Download MP4

Step 3: Host Videos on Firebase
- Upload MP4s to `web/assets/videos/`
- File naming: `explainer-professional-en.mp4`, `explainer-prospect-es.mp4`, etc.

Step 4: Embed in Websites
Professional video placement:
- `web/professionals.html`
- `web-es/professionals.html`
- `web-pt/professionals.html`
- `web-de/professionals.html`

Prospect video placement:
- `web/prospects.html`
- `web-es/prospects.html`
- `web-pt/prospects.html`
- `web-de/prospects.html`

Step 5: Track Performance (GA4)
Add events:
- `video_play_professional`
- `video_play_prospect`
- `video_complete_25/50/75/100`
